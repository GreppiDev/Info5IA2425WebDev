Map with Marker at Address

Piazza del Colosseo, 1, 00184 Roma RM, Italy

Leaflet.js Map Implementation:

This map is created using Leaflet.js with the following components:

1. Map Initialization:

  • L.map("map1").setView([41.8902, 12.4922], 15)
    • Creates a map centered on the Colosseum
    • Zoom level 15 provides city district level detail
    • See: Map initialization documentation

2. Tile Layer:

  • L.tileLayer("https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png")
    • Adds OpenStreetMap tiles as the base layer
    • {s}: Subdomain for load balancing
    • {z}: Zoom level
    • {x},{y}: Tile coordinates
    • See: TileLayer documentation

3. Marker:

  • L.marker([41.8902, 12.4922]).bindPopup(address)
    • Adds a clickable marker at the Colosseum location
    • Popup shows the full address on click
    • See: Marker documentation
    • See: Popup documentation

ESRI Satellite Map with Location Marker

ESRI Satellite Map Implementation:

This map combines Leaflet.js with ESRI's satellite imagery and adds interactive features:

1. Map Setup:

  • L.map("map2").setView([41.9022, 12.4539], 16)
    • Creates a map centered on St. Peter's Basilica
    • Zoom level 16 provides detailed building view
    • See: Leaflet Map documentation

2. ESRI Satellite Layer:

  • L.esri.basemapLayer("Imagery").addTo(map2)
    • Adds ESRI's high-resolution satellite imagery
    • Part of the esri-leaflet plugin
    • See: ESRI Basemap Layer documentation

3. Interactive Features:

  • Marker with Popup:
    • L.marker([41.9022, 12.4539]).bindPopup("St. Peter's Basilica")
    • Places a marker at the exact location
    • See: Marker documentation
  • Live Coordinates Display:
    • map2.on('moveend', function() { ... })
    • Updates coordinates when map movement ends
    • Uses .toFixed(4) for 4 decimal precision
    • See: Map Events documentation

Region Map with GeoJSON Overlay

GeoJSON Map Implementation:

This map displays Rome's historic center using GeoJSON to define the area boundary:

1. Map Setup:

  • L.map("map3").setView([41.9028, 12.4964], 13)
    • Creates a map centered on Rome's historic district
    • Zoom level 13 provides neighborhood overview
    • See: Map initialization documentation

2. GeoJSON Structure:

  • Feature Type:
    • type: "Feature" - Single GeoJSON feature
    • properties: { name: "Rome Historic Center" } - Feature metadata
    • geometry: { type: "Polygon" } - Closed shape definition
    • See: GeoJSON Format Specification
  • Coordinates Array:
    • Eight coordinate pairs forming a polygon
    • Format: [longitude, latitude] for each point
    • First and last points match to close the polygon
    • See: Leaflet GeoJSON examples

3. GeoJSON Layer:

  • L.geoJSON(historicCenterGeoJSON, { style: {...} })
    • Renders the GeoJSON polygon on the map
    • Style options:
      • color: #ff7800 (orange border)
      • weight: 3 (border thickness)
      • opacity: 0.7 (border transparency)
      • fillOpacity: 0.2 (fill transparency)
    • See: GeoJSON Layer documentation

4. Interactive Features:

  • Popup binding: .bindPopup("Rome Historic Center - UNESCO World Heritage Site")
  • Auto-zoom: map3.fitBounds(geoJsonLayer.getBounds())
    • Automatically fits the map view to the polygon boundaries
    • See: fitBounds documentation
