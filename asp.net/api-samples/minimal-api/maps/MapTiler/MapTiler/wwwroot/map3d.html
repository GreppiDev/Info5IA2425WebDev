Informazioni su Cesium e MapTiler

CesiumJS

Questa mappa 3D utilizza CesiumJS, una libreria JavaScript per la visualizzazione di dati geospaziali 3D.

MapTiler Terrain 3D

Il terrain 3D è fornito dal servizio MapTiler Terrain 3D quantized mesh, che offre dati di elevazione globale di alta qualità.

Documentazione

Elementi Cesium utilizzati in questa demo:

  • Viewer - Contenitore principale della mappa
  • TerrainProvider - Gestisce il terreno 3D di MapTiler
  • Entity - Gestisce i marker sulla mappa
  • Camera - Controlla la visualizzazione della scena
