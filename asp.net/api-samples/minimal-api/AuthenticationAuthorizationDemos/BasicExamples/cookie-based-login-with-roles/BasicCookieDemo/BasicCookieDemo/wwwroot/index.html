Cookie Authentication Demo

Esempio di autenticazione basata su cookie con ruoli e autorizzazioni in ASP.NET Core

Descrizione del Progetto

Questa applicazione dimostra come implementare l'autenticazione basata su cookie in ASP.NET Core, con supporto completo per ruoli utente e autorizzazioni granulari. Il progetto utilizza ASP.NET Core Minimal API e mostra come proteggere le risorse in base ai ruoli e alle autorizzazioni degli utenti.

Endpoint API Disponibili

L'applicazione espone i seguenti endpoint API:

Autenticazione

  • POST /login - Effettua il login
  • POST /logout - Effettua il logout
  • POST /login-with-permissions - Login con permessi speciali

Risorse Protette

  • GET /profile - Accessibile a qualsiasi utente autenticato
  • GET /admin-area - Accessibile solo agli amministratori
  • GET /power-area - Accessibile a PowerUser o Admin
  • GET /super-area - Richiede entrambi i ruoli Admin E PowerUser
  • GET /edit-content - Richiede il permesso "CanEditContent"
  • GET /my-roles - Fornisce informazioni sui ruoli dell'utente

Utility

  • GET /set-cookie - Imposta un cookie di esempio
  • GET /read-cookie - Legge il cookie di esempio
