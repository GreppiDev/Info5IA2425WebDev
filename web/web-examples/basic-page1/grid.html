CSS Grid: spiegazione semplice

Grid serve per creare layout a righe e colonne (ottimo per pagine, card, dashboard).

1) Concetto base: contenitore e celle

CSS Grid funziona quando: 1 scegli un contenitore e 2colonne e/o righe. 

/* 1) contenitore */
.grid {
  display: grid;

  /* 2) colonne */
  grid-template-columns: 1fr 1fr 1fr;

  /* spazio tra le celle */
  gap: 10px;
}

2) Colonne: grid-template-columns e repeat()

Il modo più comune per iniziare è definire le colonne con grid-template-columns.

  • 1fr 2fr: la seconda colonna è il doppio della prima
  • repeat(4, 1fr): 4 colonne uguali
  • repeat(auto-fit, minmax(140px, 1fr)): griglia responsive
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 10px;
}

3) Righe: grid-template-rows e grid-auto-rows

Puoi definire anche le righe. Se non le definisci, Grid le crea “automatiche” in base al contenuto. 

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: 60px 110px;
  gap: 10px;
}

Qui stiamo dicendo: “due righe”, la prima alta 60px e la seconda alta 110px.

4) Posizionare elementi: grid-column e grid-row

La cosa potente di Grid è che puoi far occupare a un elemento più celle: grid-column: span 2 significa “occupa 2 colonne”. 

.item-grande {
  grid-column: span 2;
}

.item-alto {
  grid-row: span 2;
}

5) Allineamenti: place-items

In Grid puoi allineare il contenuto dentro le celle. place-items è una scorciatoia per align-items + justify-items. 

.grid {
  display: grid;
  place-items: center;
}

6) Layout con grid-template-areas (molto usato)

grid-template-areas permette di descrivere un layout “a parole”, utile per strutture tipiche (header, sidebar, content, footer). 

.layout {
  display: grid;
  grid-template-columns: 220px 1fr;
  grid-template-areas:
    "header header"
    "sidebar content"
    "footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

Esercizi (veloci)

  1. In grid.css, cambia grid-template-columns della demo 3 colonne in 1fr 2fr 1fr.
  2. Nella demo responsive, prova minmax(180px, 1fr): quante colonne vedi ora?
  3. Nella demo “placement”, prova a cambiare grid-auto-rows da 70px a 90px.
