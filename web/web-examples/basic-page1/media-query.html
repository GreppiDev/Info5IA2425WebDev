Media query

Le media query permettono di applicare regole CSS diverse in base alle caratteristiche del dispositivo (es. larghezza schermo). Sono la base del responsive design.

1) La sintassi

Una media query è un “blocco” di CSS che vale solo se una condizione è vera. Esempio (mobile-first): 

@media (min-width: 760px) {
  /* regole attive da 760px in su */
  .menu { display: flex; }
}
Suggerimento: apri questa pagina e cambia la larghezza della finestra.

2) min-width vs max-width

Approccio mobile-first (consigliato)

  • Scrivi prima gli stili base per schermi piccoli.
  • Poi “aggiungi” miglioramenti con min-width.
/* base: mobile */
.card { padding: 12px; }

/* da tablet in su */
@media (min-width: 760px) {
  .card { padding: 16px; }
}

Approccio desktop-first

  • Scrivi prima gli stili per desktop.
  • Poi “riduci” con max-width per mobile.
/* base: desktop */
.nav { display: flex; }

/* sotto 760px */
@media (max-width: 759px) {
  .nav { display: none; }
}

4) Media features utili

  • (orientation: portrait) / (orientation: landscape) (orientamento)
  • (prefers-color-scheme: dark) (preferenza tema)
  • print (stampa)
/* esempio: regole diverse in stampa */
@media print {
    .site-header,
    .site-footer,
    #themeToggle { display: none; }

    body { color: #000; }
}

5) Esercizi

  1. Modifica i breakpoint in media-query.css (es. 600 / 900) e osserva cosa cambia.
  2. Fai in modo che la griglia nella demo diventi 4 colonne oltre 1200px.
  3. Cambia il colore del badge in base al breakpoint.
