Bootstrap: introduzione

Bootstrap è un framework che fornisce classi CSS pronte (layout, spacing, colori) e componenti (navbar, modal, alert…) per costruire interfacce responsive più in fretta.

Prova: riduci/allarga la finestra e osserva la navbar e la griglia qui sotto.

1) Come funziona Bootstrap (in pratica)

  • Utility classes: classi piccole e combinabili (es. p-3, mt-2, text-center, d-flex).
  • Grid system: layout responsive con .container, .row, .col. Le colonne cambiano numero e larghezza in base ai breakpoints.
  • Componenti: blocchi già progettati (navbar, cards, buttons, alerts…).
  • JS per componenti: alcune parti (collapse, modal, dropdown) funzionano con attributi data-bs-* senza scrivere JS manuale.

2) Breakpoints (responsive)

Bootstrap usa breakpoints standard (mobile-first). I più comuni: sm (≥576px), md (≥768px), lg (≥992px), xl (≥1200px).

Esempio: navbar-expand-md
La navbar è “compressa” (hamburger) sotto md, espansa da md in su.
Esempio: col-12 col-sm-6 col-lg-3
1 colonna su mobile, 2 su sm, 4 su lg.

4) Utility classes: spacing e allineamenti

Bootstrap offre moltissime utility: margini/padding (m-*, p-*), flex (d-flex, justify-content-between), testo (text-center), ecc.

Esempio flex d-flex
Nessun CSS scritto: solo classi.
Esempio spacing
Prova a cambiare p-3 in p-1 o p-5 nel codice.
