Flexbox CSS: spiegazione semplice

Flexbox serve per disporre elementi in riga o colonna e gestire spazi/allineamenti.

1) Concetto base: contenitore e elementi

Flexbox funziona quando: 1 scegli un contenitore e 2 metti dentro degli elementi. Poi attivi Flexbox sul contenitore con display: flex. 

/* 1) contenitore */
.container-flex {
  display: flex;
}

/* 2) elementi dentro il contenitore */
.container-flex .item {
  /* stile dell'item */
}

Senza Flexbox gli elementi andrebbero uno sotto l’altro; con Flexbox vanno in riga.

2) Gli assi: principale e trasversale

Flexbox ragiona su due assi:

  • Asse principale: dipende da flex-direction (riga o colonna).
  • Asse trasversale: è l’asse “perpendicolare” a quello principale.

flex-direction: row (default)

.box {
  display: flex;
  flex-direction: row;
}

3) Spaziatura: gap

gap imposta lo spazio tra gli item (molto comodo: evita margin “a mano”). 

.box {
  display: flex;
  gap: 10px;
}

4) Allineamenti: justify-content e align-items

Le due proprietà più usate per allineare sono:

  • justify-content: allinea lungo l’asse principale.
  • align-items: allinea lungo l’asse trasversale (tutti gli item insieme).

Centrato (justify-content + align-items)

.box {
  display: flex;
  justify-content: center;
  align-items: center;
}

5) Andare a capo: flex-wrap

Se hai tanti elementi e vuoi che vadano a capo quando non c’è spazio, usa flex-wrap: wrap. 

.box {
  display: flex;
  flex-wrap: wrap;
  gap: 10px;
}

6) Far crescere gli elementi: flex (grow/shrink/basis)

Per distribuire lo spazio, spesso si usa la scorciatoia flex sugli item. Un caso tipico: “questi elementi devono occupare lo stesso spazio”. 

/* Tutti gli item crescono in modo uguale */
.item {
  flex: 1;
}

/* Un item può restare della sua dimensione */
.item-no-grow {
  flex: 0 0 auto;
}
